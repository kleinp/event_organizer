# Event Organizer

## Installation

- Install node 8.11LTS from: https://nodejs.org/en/
- Install git from: https://git-scm.com/
    - Be sure to install git bash
- Check out files
    `git clone http....`
- Navigate to folder
    `cd /path/to/folder/`
- Install dependencies
    `npm install`

## Running

`node index.js`
